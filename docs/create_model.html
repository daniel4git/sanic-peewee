创建model¶

不同于peewee或者peewee-async,我们创建的model继承自与app绑定后的Peewee对象下的类`AsyncModel`. 它已经绑定了内部类中的database为app的连接数据库.

: 绑定了数据库:

class Meta:
    database = db

但定义方式与peewee完全一样,使用相同的Field.

一个典型的model如下:

peewee = Peewee(dburl)
db = peewee(app)
class KeyValue(db.AsyncModel):
    key = CharField(max_length=40, unique=True)
    text = TextField(default='')
